## 如何更新

一如既往，您可以使用以下命令之一来更新您的机器人：

### docker-compose

```bash
docker-compose pull
docker-compose up -d
```

### 通过安装脚本安装

``` bash
# 退出虚拟环境 (venv) 并运行：
./setup.sh --update
```

### 普通原生安装

``` bash
git pull
pip install -U -r requirements.txt
```
